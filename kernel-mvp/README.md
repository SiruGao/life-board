# 内核 Kernel MVP

一个本地优先、不会替用户作出价值判断的苏格拉底式思考训练工具。

## 首版闭环

1. 用户先写下问题、初始立场、依据与确信程度。
2. 本地规则引擎识别绝对化表达、外部评价牵引、判断外包、紧迫性放大等线索。
3. 系统按概念、证据、可证伪性、最强反方、价值、后果和责任生成结构化追问。
4. 用户逐题回答后，用自己的语言形成暂时判断、下一步行动和愿意承担的代价。
5. 浏览器本地保存纵向档案，展示重复出现的思考线索；用户可导出、导入和彻底删除。

## 产品原则

- 人的初步判断在前，系统追问在后。
- 纠正事实与推理错误，但不替用户裁决价值框架。
- 最终结论必须回到用户自己的语言中。
- 记忆可见、可导出、可删除，不做广告画像。
- 成功指标是用户逐渐能够独立提出反例和承担判断，而不是对话时长。

## 运行

这是一个无依赖的静态应用。直接打开 `index.html`，或在目录中运行：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173`。

## 当前边界

首版使用本地规则引擎，不调用外部大模型。这保证部署后立即可用并避免浏览器暴露 API Key。下一阶段应增加服务端模型代理、可验证的哲学知识库、反迎合评估和浏览器扩展入口。
