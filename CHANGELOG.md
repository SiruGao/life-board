## [0.6.0] - 2026-06-18

### Added

- 完成审议后的艺术化判断卡；
- 单条审议 Markdown、纯文本和 JSON 私密导出；
- 默认去隐私的公开分享摘要；
- 问题、当前判断和行动必须主动勾选才会进入分享内容；
- 七天后重新检验判断的复查提示；
- 本地版与 AI 版五分钟快速开始；
- 一键 Vercel 部署入口；
- 公开发布与 GitHub Star 增长计划；
- 判断结果和隐私分享专项测试。

### Changed

- README 改为“立即体验 → 一键部署 → 能力 → 隐私 → 贡献”的公开发布结构；
- Bug Report 覆盖判断卡、Gateway、AI、PWA、部署和隐私场景；
- PWA 离线缓存升级为 v6。

### Privacy

- 结果导出全部在浏览器本地执行，不新增网络请求；
- 公开分享默认不包含审议问题、结论和下一步行动；
- 完整导出明确标记为私密档案。

## [0.5.2] - 2026-06-18

- 高级连接设置折叠区；
- 令牌、供应商、超时、余额和模型格式的可读错误提示。

## [0.5.1] - 2026-06-18

- 触摸设备与减少动态效果完整降级。

## [0.5.0] - 2026-06-18

## [0.3.0] - 2026-06-18

### Added

## [0.2.0] - 2026-06-18

### Added

- 五种审议模式：决策、观点、阅读、自我反思和 AI 使用审计；
- 绝对化表达、外部评价牵引、紧迫性放大、判断外包和流畅性信任线索；
- 结构化逐题追问；
- 最终判断、下一步行动和责任确认；
- 审议前后确信程度比较；
- 长期思考档案和重复线索统计；
- 单条删除、全部删除、JSON 导入与导出；
- PWA 安装和离线缓存；
- GitHub Pages、Vercel 和 Netlify 静态构建支持；
- 产品原则、架构、路线图、贡献指南和安全政策；
- 中英文 GitHub 项目首页。

### Changed

- 将原仓库重构为单一 Socratic Kernel 产品；
- 应用从子目录迁移到仓库根目录；
- 统一产品名称、存储键、部署脚本和文档；
- GitHub 首页升级为产品化展示结构。

### Removed

- 原 LifeBoard React/Vite 应用；
- 不再需要的 TypeScript、ESLint 和 Vite 配置；
- 旧的部署说明和临时 MVP 副本。
