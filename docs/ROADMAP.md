# Roadmap

## V0.3 — 首次体验与增长基础

- 五种审议模式：决策、观点、阅读、自我反思、AI 使用审计
- 本地认知线索检测与结构化苏格拉底式追问
- 首次访问引导和可编辑示例
- 首次四问轻量流程
- 最终判断与责任确认
- 长期思考档案
- 数据导入、导出和删除
- PWA 安装与离线使用
- Pull Request 自动测试与构建

## V0.4 — 可替换模型层

- 服务端模型代理，浏览器不保存供应商密钥
- 支持 OpenAI、Anthropic、Gemini、DeepSeek、Qwen、Kimi 和 xAI
- 用户主动选择本地规则或 AI 增强
- AI 请求失败时自动保留本地问题
- 默认不发送历史档案、后续回答和长期模式统计
- Vercel Serverless Functions 与跨域网关配置
- Beta 访问令牌、请求大小限制和超时控制
- 下一步：用户认证、订阅额度、分布式限流和成本控制

## V0.5 — 浏览器介入层

- Chrome、Edge、Firefox 扩展
- 选中文字后启动“阅读质疑”
- 在 AI 网页中识别用户是否正在外包价值判断
- 仅在高价值节点提示，不持续打扰
- 网页内容默认不上传，用户主动确认后才进入审议

## V0.6 — 桌面与移动应用

- 抽取共享核心包
- Tauri：macOS、Windows 和 Linux 桌面端
- Capacitor：iOS 和 Android 移动端
- 系统分享菜单、全局快捷键和本地通知
- 本地加密数据库与安全令牌存储
- App Store / Google Play 合规与订阅恢复

## V0.7 — 哲学知识层与反迎合审查

- 第一模型生成问题，第二审查器检测迎合、偷换概念和虚假确定性
- 模型输出与本地规则发生冲突时，展示冲突而不是静默覆盖
- 哲学原典与可靠二手资料的可追溯检索
- 将苏格拉底、亚里士多德、康德、庄子、阿伦特、福柯等思想转换为明确审议协议
- 区分事实检验、逻辑检验和价值视角
- 禁止用“哲学家人格模仿”替代来源和论证

## V0.8 — 个人认知模型

- 用户可见、可编辑的思考模式图谱
- 为每条推断显示证据、置信度和反例
- 支持“这不是我”反馈与记忆撤销
- 独立性指标：先行立场率、主动反例率、判断外包率、无 AI 完成率
- 端到端加密的可选多设备同步

## 发布前门槛

- 完成隐私威胁模型
- 完成反迎合基准测试
- 明确非心理治疗边界
- 建立数据删除与迁移测试
- 建立认证、额度、限流和供应商成本告警
- 不使用对话时长或依赖程度作为核心增长指标
